C is a functional/procedural oriented programming

C++ is a Multi paradigm language

Object Oriented programming was introduced because programming in C using functions was 
a headache, there were many errors while coding and that cost millions of dollars so to
solve that issue C++ with its Object Oriented Programming was introduced. Object Oriented 
Programming is a high level concept so its easy for anyone to write meaningful code with ease.

Object Oriented programming has also bought many features
  
  1) Data Abstraction and Encapsulation
  2) Inheritance
  3) Polymorphism
  4) Reduction of redundancy
